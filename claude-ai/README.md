# claude.ai — Student Internship Strategist

Install the skill file and Claude becomes your internship strategist in any conversation.
No setup. No code. Works on the free plan.

---

## Install

### Method 1 — Settings (Recommended)

1. Go to [claude.ai](https://claude.ai)
2. Open **Settings** → **Skills**
3. Click **Install from file**
4. Upload `student-internship-strategist-SA.md`

### Method 2 — Drag and Drop

1. Open any conversation on claude.ai
2. Drag `student-internship-strategist-SA.md` into the chat
3. Claude will detect it as a skill file and install it

---

## Using It

Once installed, start any conversation with something like:

- *"I need help finding a tech internship in Saudi Arabia"*
- *"Can you review my CV for a software engineering role?"*
- *"I want to apply to SDAIA — help me prepare"*
- *"Write me a cold message to the CTO of Lean"*
- *"Find me open AI internships in Riyadh"*

Claude will automatically use the skill when it detects internship or career intent.

## Connectors That Help

Connect these in claude.ai Settings → Connectors for the best experience:

| Connector | What it adds |
|---|---|
| **Indeed** | Live job search built into the conversation |
| **ZipRecruiter** | Secondary job search, international roles |
| **Notion** | Automatic pipeline tracking in your workspace |
| **Google Drive** | Sync CVs and documents across devices |

Connectors are optional. The skill works without them — it uses web search
as a fallback for job discovery.

---

## Tips

- Upload your CV as a PDF at the start of the session
- Export your LinkedIn connections as a CSV and upload it — Claude will
  cross-reference your network against target companies
- Be specific in your answers. The more detail you give, the better
  everything Claude produces for you
