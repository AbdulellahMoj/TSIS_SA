# Contributing

This repository is built for Saudi tech students.
If you've used it, found something missing, or have something that worked — contribute it.

---

## What's Useful to Add

**Saudi company careers URLs**
If you find a company's careers page that isn't in `references/saudi_companies.md`,
add it with the correct ATS platform noted.

**Cold messages that got replies**
Anonymised versions of messages that actually worked.
Add them to `templates/cold_messages.md` with a note on the context.

**ATS patterns**
If you encounter a Saudi-specific ATS platform not listed in `references/ats_patterns.md`,
document its URL pattern and common field names.

**Question bank additions**
Good discovery questions for specific fields (cybersecurity, cloud, hardware, consulting).
Add to the relevant phase in `references/question_bank.md`.

**Script improvements**
Bug fixes, better parsing, new platforms for `job_search.py`.

---

## How to Contribute

1. Fork the repo
2. Create a branch: `git checkout -b add/company-list-update`
3. Make your changes
4. Submit a pull request with a short description of what you added and why

---

## What Not to Contribute

- Personal information (CVs, names, emails, phone numbers)
- Fabricated job listings or fake data
- Content from other people without permission
- Anything behind a paywall or requiring login to access

---

## Questions

Open an issue if something is broken, missing, or unclear.
